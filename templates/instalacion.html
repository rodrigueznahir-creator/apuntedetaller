INSTALACIÓN

Flask es un microframework de desarrollo web para Python, que proporciona herramientas básicas y flexibilidad para crear aplicaciones web rápidamente. Se caracteriza por ser ligero y minimalista, permitiendo a los desarrolladores elegir las extensiones y bibliotecas necesarias para sus proyectos.

1. Crear un entorno virtual

El entorno virtual permite aislar las dependencias del proyecto para evitar conflictos con otras instalaciones de Python.

python -m venv venv

Esto creará una carpeta llamada venv que contendrá el entorno virtual.

2. Activar el entorno virtual

En Windows:

venv\Scripts\activate

En Linux / Mac:

source venv/bin/activate

Al activarlo, la consola mostrará el nombre del entorno virtual al inicio de la línea.

3. Instalar Flask

Con el entorno virtual activado, instalamos Flask con pip:

pip install flask

Podés verificar la instalación usando:

pip show flask
